# Iris Flower Classification

Machine learning model to classify iris flowers into species based on petal and sepal features.

## Results

| Model | Accuracy |
|-------|----------|
| Logistic Regression | 93.33% |
| Decision Tree | 93.33% |
| Random Forest | 90.00% |
| **Support Vector Machine** | **96.67%** |
| K-Nearest Neighbors | 93.33% |

**Best Model:** Support Vector Machine with **96.67% accuracy**

## How to Run

pip install -r requirements.txt
python iris_classification.py
